# end points

---

/species ->
	/angels -> returns a JSON object containing the names and data of all the angel characters.
	/demons -> Returns a JSON object containing the names and data of all the demon characters.

---

/characters ->
	/ -> Returns a JSON object containing the names and data of all characters.
	/:character_name -> returns a JSON object containing the name and data of the specified character; if not found, returns a 404 error, not found.
